### Create a project, deploy it to GitHub and maintain it.


#### **Objectives**

- Learn GitHub.
- Create a project.
- Upload it to GitHub.
- Maintain it.

#### **Tips**

- Maintaining a good project will gain you more points.
- Searching on Google will help you for this task.
